# Automation Portfolio – Hua Cao

This portfolio showcases my experience in building scalable automation solutions across backend, web, mobile, and performance testing using Playwright, WebdriverIO, Appium, Postman, and K6.


---

## 🔹 API Automation – User Deactivation

🎥 [Watch Demo](https://drive.google.com/file/d/1IzU9azkpravgajN0cKD9INqp_eHVC13N/view?usp=drive_link)

## 🔹 Performance Testing – K6

📄 [View Report](https://drive.google.com/file/d/1PjGwO58rS5IWyjUdSQJlMAY-dfkYrPI6/view?usp=drive_link)
